Professional 오픈소스 협업훈련, 리얼 Git반 2주 과정
(5회 / 월,수,월,수,금: 19:30-22:30 / 30명 정원)

단순히 Github에 소스코드를 올렸다고 오픈소스 프로젝트라고 생각하진않습니다.
Professional 하게 협업하는게 중요하다고 생각합니다. 오픈소스 개발방식에 기반해 Git/Github를 중고급수준으로 훈련하신다면 개발자로서 한단계 더 성장하실수있는 시간을 될거라고 생각합니다.

Git 이나 Github를 사용하는데는 문제없지만
해외 오픈소스 개발자들과같이 professional 하게 협업하는 방법을 배우고 싶으신분들을 위한 반입니다. 예를 들어 협업할때 아주 자주,유용하게 쓰이는 rebase 나 rebase --interactive 기능은 왜 쓰는것이고 구체적으로 어떤 상황에서 쓰게되는지 ?, commit하나의 크기는 어느정도로 하는게 적합한지 ?, commit 메시지는 항상 한줄로만 적어왔는데 해외 유명개발자들은 어떻게 작성하는지 ? 등 중고급수준의 미션/예제들로 실습을 진행합니다. 마지막 회차에는 실제 원하는 오픈소스 프로젝트에 작은 기여(contirbution)도 진행합니다.
