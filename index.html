리얼리눅스 수강생들이 전하는 생생한 후기들

SW 기본반

수강생 후기

대학교 수업에서는 암기식으로 공부해서 그런지 시간이 지나면 기억기 잘나지 않습니다. 그래서 수업을 듣게 되었고 reallinux에서 실습위주의 수업이 개념을 다시 잡는데 도움이 되었고 기억에도 잘 남을 것 같습니다.

Try 오컨

수강생 후기

소수정예라서 더 얻어 가는데 많았던것 같습니다. 오픈소스에 기여 할 수 있다는 자신가을 얻고 여러 경험을 해 보는 부분에서 많이 배워갑니다. 좋은 사람들을 만나서 많이 도움을 받아 해낼 수 있었던 점도 좋았습니다. 앞으로 배운걸 까먹지 않고 잘 활용 할 수 있도록 컨트리뷰톤을 이어서 진행하려고 합니다.

OS중급A반

수강생 후기

현업에서는 공부하기 힘든 부분에 대하여 공부하였으며 커널 뿐만 아닌 운영 시 실제 유용한 명령어에 대하여 알수 있어서 좋았습니다. 커널 또한 전체적으로 그림이 보이니 Top Down으로 진행하는 좋은 수업이었습니다.

OS중급B반

수강생 후기

평소에 책으로 개념만 공부한 리눅스를 실제 코드로 보면서 어떻게 동작하는지 알고 싶었습니다. 시중에 책도 없고 강의도 없어서 언젠가는 이런 강의가 나오겠지 했는데 듣게 되어서 다행이라고 생각합니다.

리얼리눅스 수강생들의 이야기

일주일에 1번 수업이 직장인 입장에서 부담없고 충분히 공부할 시간도 확보 할 수 있어서 좋았다. 판서와 이론 & 실제 코드로 확인하는 과정도 매우 좋았다.

reallinux 수강생

contribution에 필요한 git 명령어를 실습을 통해 확실히 익힐 수 있었고, 단계별로 contribution하는 방법을 배 울 수 있었다. 수업이 끝나더라고 혼자서 관심있는 프로젝트에 contribution 할 수 있을 것 같다.

reallinux 수강생

전반적으로 좋았습니다. 스스로 과제를 부여하고 진행하는 과정이 익수치 않아서 적응하는데는 시간이 필요했습니다만, 개발자로써는 필요한 능력을 기를수 있었던 과정이라 생각합니다.

reallinux 수강생

머리 속에 개념만으로만 남아있던 부분들을 다시 배우고 실습을 통해서 더욱 깊이 이해할수 있었습니다. 기본을 다지기에 좋은 만족스러운 강의였습니다.

reallinux 수강생

소스코드만 읽는 방식이 책으로 공부할 때는 와 닿지 않는 부분이 있었는데 트레이싱을 이용해 분석을 하니 이해하기 한결 수월했습니다.
